Projeto

Este projeto foi desenvolvido para avaliação final no curso de Especialista em Cibersegurança, ministrado pela ANCIBER – Academia Nacional de Cibersegurança. O principal objetivo é demonstrar as competências adquiridas ao longo da formação.
A aplicação é desenvolvida em Python, destina-se exclusivamente para fins académicos e deve ser utilizada apenas dentro dos limites legais e éticos da cibersegurança.

Preparação

Antes de executar o projeto, deve instalar as seguintes bibliotecas python:

pip install tkinterweb
pip install Werkzeug
pip install Flask

Utilização

Após executar a aplicação, tem disponível as seguintes funcionalidades:

  • Ativar Servidor Web - inicia o serviço web na sua máquina e fica disponível no endereço http://localhost:5000 ou http://IP_LOCAL:5000 do seu browser. Este serviço permite-lhe que a sua máquina fique a observar os utilizadores que clicam nos links enviados por e-mail.
    Nota: os links apenas podem ser acedidos se o destinatário se encontrar na mesma rede;
  • Configurar SMTP - configuração do serviço SMTP, para que seja possível efetuar o envio de e-mails.
    Nota: a configuração é eliminada após o fecho da aplicação;
  • Templates - consultar, editar e eliminar registos, associados aos conteúdos dos e-mails que são enviados;
  • Credenciais - consultar se o link foi aberto, se as credenciais foram inseridas e outras informações, acerca de cada e-mail enviado. Possibilidade de copiar e exportar os resultados obtidos;
  • Enviar E-mail - enviar e-mail com a respetiva página falsa e conteúdo escolhido. Opção disponível se serviço web ativo;

Versões

Versão 1.0 ( fevereiro 2025 )
Possíveis melhorias em futuras versões:

  • Criação de listas/grupos de pessoas com detalhes importantes (exemplo: departamento);
  • Criação de campanhas, com indicação do estado, agendamento e listas/grupos de pessoas destinadas;
  • Criação de dashboards, onde seja possível efetuar análises como, utilizadores com maior risco de phishing;
  • Criação de novas páginas falsas de inicio de sessão (exemplo: gmail, facebook, instagram);
  • Envio de e-mails em massa, através de ficheiro TXT e/ou CSV;
  • Envio de e-mails por API (exemplo: Microsoft Graph);
